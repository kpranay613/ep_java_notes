
Java:
=====
It is High level language created by using c-lang.
It uses OOPs concepts , also known as Object Oriented Programming lang.
It is Open Source
It is Secure
It is Robust
It is Multi-tasking
It is Platform Independent
It is used to develop Desktop Applications , Web Applications , Mobile Applications
It is mostly used for Server Side Programming. (Backend Programming)
It is API (Application Programming Interface)
Technologies:
AI , ML , Devops Tools (Jenkis,SonarQube,Jira), Microservices , Blockchain , Cloud Computing
Pega , Salesforce , SAP , HANA , Hadoop , Mulesoft , Dockers (Kubernetes) , IOT , Android

Types of Java Editions:
-----------------------

1. JSE : Java Standard Edition (core java)
         Desktop Application and Standalone Application

	Ex: Paint , Eclipse , Games , Browser , Tools  , Software in electronic devices 

2. JEE : Java Enterprise Edition (advance java)

	 Web Applications and Enterprise Applications

	Web Ex: University Website , State Govt Website 
        Enterprise Ex:  Banking Apps/Server , Google , Amazon , Facebook , Youtube ..
	

3. JME : Java Micro(Mobile) Edition 

 Ex:  Mobiles Apps , Application for small devices , Video Games
